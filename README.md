# Список задач на чистом JavaScript

Небольшое, лаконичное приложение для управления задачами, созданное без использования сторонних библиотек и фреймворков — исключительно HTML, CSS и нативный JavaScript.

Основная идея проекта — продемонстрировать создание полноценного интерактивного веб-приложения, опираясь лишь на встроенные возможности браузера. Материал будет полезен тем, кто осваивает-разработку и хочет глубже разобраться во взаимодействии с DOM-деревом и механизмах работы localStorage.

🧰 Используемые технологии
HTML5 — базовая структура страницы

CSS3 — стилизация и адаптивность

JavaScript (ES6+) — вся логика приложения
Проект не использует сборщики модулей, дополнительные библиотеки или фреймворки — только чистый код.
